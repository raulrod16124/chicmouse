# ChicMouse Studio — Design System & Brand Guidelines

---

## Brand Essence

ChicMouse Studio representa:

- Creatividad estructurada
- Innovación en juegos
- Simplicidad elegante

**Concepto central:** “Idea Factory” — transformar ideas simples en experiencias de juego.

**Personalidad de marca:**

- Moderna
- Minimalista
- Inteligente
- Cercana (sin ser infantil)

---

## Logo

**Concepto:** Monograma CM integrado en la silueta de un ratón. El ratón se reconoce primero y las letras se descubren después.

**Características:**

- Geométrico con equilibrio orgánico
- Trazo medio (ni fino ni grueso)
- La C tiene ligera curva (cuerpo del ratón)
- La M es más estructurada (contraste)
- Punto amarillo como ojo / idea

**Uso correcto:**

- Centrado
- Sin texto dentro del icono
- Mantener proporciones originales
- Usar versión adecuada según fondo

**Uso incorrecto:**

- No deformar proporciones
- No cambiar colores arbitrariamente
- No añadir sombras ni efectos
- No usar fondos complejos

## Color System

### Primary — Dark Mode (principal)

| Token                       | Valor     |
| --------------------------- | --------- |
| Background principal        | `#0B132B` |
| Background secundario       | `#1C2541` |
| Superficie                  | `#243B53` |
| Texto principal             | `#F5F5F5` |
| Texto secundario            | `#C9D1D9` |
| Acento principal (idea/ojo) | `#FFC857` |
| Acento secundario           | `#3A86FF` |

**Uso:** Web principal · Branding · Google Play / App Store · Producto

### Secondary — Light Mode (complementario)

| Token                 | Valor     |
| --------------------- | --------- |
| Background principal  | `#F7F9FC` |
| Background secundario | `#FFFFFF` |
| Texto principal       | `#0B132B` |
| Texto secundario      | `#4A5568` |
| Acento principal      | `#FFC857` |
| Acento secundario     | `#3A86FF` |

**Uso:** Landing pages · Documentación · Marketing

### Reglas de color

- El amarillo (`#FFC857`) es solo acento — no usar en grandes superficies
- Evitar negro puro (`#000000`)
- Evitar blanco puro (`#FFFFFF`) en modo oscuro
- No usar más de dos colores de acento simultáneamente

---

## Tipografía

**Fuentes recomendadas:** Inter · Manrope · General Sans

### Jerarquía

| Nivel | Peso     | Tamaño  |
| ----- | -------- | -------- |
| H1    | Bold     | 32–40px  |
| H2    | SemiBold | 24–32px  |
| H3    | Medium   | 18–24px  |
| Body  | Regular  | 14–16px  |

**Estilo:** Limpio · Sin adornos · Espaciado amplio

---

## UI Components

### Botones

**Primary**

- Fondo: `#FFC857`
- Texto: `#0B132B`
- Hover: versión ligeramente más oscura

**Secondary**

- Fondo oscuro o transparente
- Texto claro

### Cards

- Fondo: `#1C2541`
- Border radius: `12px`
- Padding: `16px`
- Sin bordes duros
- Sombra sutil opcional

### Links

- Color: `#3A86FF`
- Hover: más brillante o subrayado

---

## Visual Language

**Estilo general:**

- Flat o semi-flat
- Vector 2D
- Limpio y geométrico

**Elementos clave:**

- Puntos (referencia al ojo del ratón)
- Líneas curvas tipo “cola”
- Formas simples

---

## UX & Layout

**Principios:**

- Uso generoso de espacio negativo
- Jerarquía clara
- Evitar saturación

**Layout:**

- Grid consistente
- Secciones bien separadas
- Alternancia de fondos

**Microinteracciones:**

- Hover suave en botones
- Transiciones rápidas (150–250ms)
- Feedback visual claro

---

## Anti-patterns

Evitar:

- Estilo cartoon en la marca (solo en los juegos)
- Uso excesivo de colores
- Gradientes sin control
- Interfaces recargadas
- Iconografía inconsistente

---

## Brand Strategy

**Diferenciación clave:**

| Marca       | Juego     |
| ----------- | --------- |
| Elegante    | Divertido |
| Minimalista | Colorido  |
| Profesional | Casual    |

> **Regla clave:** Nunca mezclar el estilo visual de marca con el del juego.

---

## Aplicación

**Google Play:**

- Usar versión dark del icono
- Sin texto
- Alto contraste

**Web:**

- Dark mode como principal
- Light mode opcional

**Redes:**

- Uso consistente del icono
- Fondos limpios

---

## Evolución futura

Posibles extensiones:

- Sistema de ilustraciones
- Animaciones de marca
- Design tokens
- UI kit reusable

---

## Resumen

ChicMouse Studio debe percibirse como una marca moderna, limpia y creativa que construye juegos con inteligencia y estilo.
