## About ChicMouse Studio

**ChicMouse Studio** is an independent development studio focused on creating polished mobile games and applications. This repository contains the source code for our official showcase website — the central hub where visitors can discover our products, read about our team, and get in touch.

The platform is designed with a dark, modern aesthetic and supports both English and Spanish, reflecting our commitment to reaching players worldwide.

---

## 🚀 Tech Stack

| Layer                    | Technology                                                                         |
| ------------------------ | ---------------------------------------------------------------------------------- |
| **UI Framework**         | [React 18](https://reactjs.org/) + [TypeScript 5](https://www.typescriptlang.org/) |
| **Build Tool**           | [Vite 4](https://vitejs.dev/)                                                      |
| **Styling**              | [styled-components v6](https://styled-components.com/)                             |
| **Animations**           | [Framer Motion](https://www.framer.motion.com/)                                    |
| **Icons**                | [Lucide React](https://lucide.dev/)                                                |
| **Internationalisation** | [react-intl](https://formatjs.io/docs/react-intl/) (EN / ES)                       |
| **Forms**                | [Formik](https://formik.org/) + [Yup](https://github.com/jquense/yup)              |
| **Routing**              | [React Router v6](https://reactrouter.com/)                                        |
| **Backend (contact)**    | [Express](https://expressjs.com/) + [Nodemailer](https://nodemailer.com/)          |
| **Database & Hosting**   | [Firebase 10](https://firebase.google.com/)                                        |
| **Testing**              | [Vitest](https://vitest.dev/) + [Testing Library](https://testing-library.com/)    |

## 🌐 What's on the site

- **Home** — Studio introduction with hero section and product spotlight.
- **Apps** — Individual pages for each ChicMouse application with screenshots and details.
- **About Us** — Meet the team behind ChicMouse Studio.
- **Contact** — Reach us directly through the built-in contact form.
- **Legal** — Privacy Policy and Terms & Conditions for each product, available in English and Spanish.
